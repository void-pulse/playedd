# Tags

Tags are a SEARCH signal. Fill them with what people actually type, not with branding.

## Philosophy
- **Search intent first.** Every tag should be a phrase a real person would type into the search bar. ~12-15 tags per video.
- **Lead with the video's main keyword phrase** (the core thing it's about), then supporting long-tail variations.
- **Brand tags are LOW priority, not the lever.** Keep only the last 1-2 slots for them. They don't drive discovery; search-intent tags do.

## Brand tags (the last 1-2 only)

Playedd, how the world plays you, hidden business models

## Category

Channel default category = Education. Leave the Education academic sub-fields (Type, Problems, Academic system, Level, Exam/course/standard) blank — they're for formal coursework, not explainers.
